## How to use:
*Need Git pkg*
```
pkg install git
```
*Clone Repositories*

```
git clone https://github.com/oxygenfox/termux-setup.git
```
*Move to directory*

```
cd /termux-setup
```
*Execute tools*

```
chmod +x tech.sh
```
```
./tech.sh
```
or
```
sh tech.sh
```
